# 🔍 搜索功能说明

## 功能概述

建辉慈善网站的搜索功能已经完整实现，用户可以快速搜索文章、项目和合作伙伴信息。

## 访问方式

### 1. 头部搜索按钮
点击网站头部的搜索图标 📍，直接跳转到搜索页面

### 2. 直接访问
在浏览器中访问：`http://localhost:3100/search`

### 3. URL搜索
可以直接通过URL带参数访问：
```
http://localhost:3100/search?q=慈善
```

## 功能特性

### ✨ 核心功能

1. **多类型搜索**
   - 文章搜索
   - 项目搜索
   - 合作伙伴搜索

2. **搜索筛选**
   - 可选择搜索类型（文章/项目/合作伙伴）
   - 支持组合搜索
   - 快速切换搜索范围

3. **智能结果展示**
   - 按类型分组显示结果
   - 关键词高亮显示
   - 显示相关元数据（日期、分类等）
   - 封面图预览

4. **无结果优化**
   - 友好的空状态提示
   - 搜索建议
   - 快速链接推荐

### 🎨 用户体验

1. **热门搜索推荐**
   - 未搜索时显示热门关键词
   - 快速链接导航
   - 一键快速搜索

2. **响应式设计**
   - 支持桌面端和移动端
   - 自适应布局
   - 优化的触摸交互

3. **加载状态**
   - 骨架屏加载效果
   - 平滑的过渡动画
   - 清晰的加载反馈

## 技术实现

### 前端技术
- Vue 3 Composition API
- TypeScript
- Element Plus UI
- Vue Router

### API集成
- 使用现有的API端点：
  - `getArticles()` - 文章搜索
  - `getProjects()` - 项目搜索
  - `getPartners()` - 合作伙伴搜索

### 性能优化
- 并行请求多个API
- 结果分页加载
- 图片懒加载
- 响应数据缓存

## 使用示例

### 搜索"慈善"
1. 在搜索框输入"慈善"
2. 点击搜索按钮或按回车
3. 查看分组结果
4. 点击任意结果查看详情

### 快速搜索
1. 在搜索页面点击热门搜索标签
2. 自动执行搜索
3. 浏览结果

### 类型筛选
1. 勾选或取消勾选搜索类型
2. 自动重新搜索
3. 更新结果展示

## URL结构

```
/search?q=关键词
```

示例：
- `/search?q=慈善` - 搜索"慈善"
- `/search?q=助学` - 搜索"助学"
- `/search` - 搜索页面首页
