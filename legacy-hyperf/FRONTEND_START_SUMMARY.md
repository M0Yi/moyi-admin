## 🚀 如何使用


## 🔌 API 集成

所有 API 请求都已配置好，会自动代理到后端：
- 开发环境: `http://localhost:6501/api/v1`
- 生产环境: `/api/v1`（相对路径）

### API 模块
- `projectsApi` - 项目接口
- `articlesApi` - 文章接口
- `donationsApi` - 捐赠接口
- `storiesApi` - 故事接口
- `statsApi` - 统计接口
- `searchApi` - 搜索接口
- `commonApi` - 通用数据接口
