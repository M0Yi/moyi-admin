# 移动端首页设计说明

## 设计理念

移动端首页采用与桌面端一致的布局结构和导航方式，只对内容模块进行移动端适配，而非采用APP样式的底部导航栏。

## 核心特性

### 1. **一致的导航体验**
- ✅ 使用顶部导航栏（与桌面端相同）
- ✅ 使用完整页脚（与桌面端相同）
- ✅ 保持网站统一的视觉风格

### 2. **响应式内容布局**
- 轮播图：50vh高度（移动端优化）
- 统计卡片：悬浮卡片设计，触控友好
- 新闻列表：单列卡片布局
- 捐赠CTA：全宽渐变背景

### 3. **自动设备检测**
- 移动设备访问 `/` → 自动跳转 `/m`
- 桌面设备访问 `/m` → 自动跳转 `/`
- 基于User-Agent和屏幕宽度双重判断

## 页面结构

```
移动端首页 (/m)
├── 顶部导航栏（AppHeader）
├── Hero Section
│   ├── 轮播图（50vh）
│   └── 统计数据悬浮卡片
├── 新闻中心
│   ├── 分类标签（横向滚动）
│   └── 新闻卡片网格（单列）
├── 捐赠CTA（全宽背景）
└── 页脚（Footer）
```

## 主要模块

### Hero Section
- **轮播图**：50vh高度，移动端优化
- **统计卡片**：悬浮设计，显示累计捐赠和捐款人次
- **图标**：使用Icon组件，专业简洁

### 新闻中心
- **分类标签**：横向滚动，点击切换
- **新闻卡片**：
  - 大图展示（180px高度）
  - 标题、摘要、日期
  - 分类标签（左上角）
  - 点击跳转详情页

### 捐赠CTA
- 渐变背景（#ff6b35 → #ff5000）
- 标题、描述、按钮居中对齐
- 全宽布局，视觉冲击力强

## 样式特点

### 配色方案
- 主色：#ff5000（品牌橙）
- 背景色：#f5f7fa（浅灰）
- 卡片背景：#fff（白色）
- 文字色：#333（深灰）

### 间距规范
- Section间距：20px
- 卡片间距：20px
- 内边距：16px
- 标题间距：24px

### 圆角规范
- 卡片：12px
- 按钮：24px
- 标签：24px

### 阴影规范
- 卡片：0 2px 12px rgba(0, 0, 0, 0.08)
- 统计卡片：0 4px 20px rgba(0, 0, 0, 0.1)

## 响应式断点

### 移动设备（≤ 768px）
- 访问 `/` 自动跳转 `/m`
- 使用移动端布局

### 桌面设备（> 768px）
- 访问 `/m` 自动跳转 `/`
- 使用桌面端布局

## 技术实现

### 设备检测
```typescript
import { isMobileDevice, isMobileWidth } from '@/utils/device'

const isMobile = isMobileDevice() || isMobileWidth()
```

### 路由配置
```typescript
// 移动端首页
{
  path: '/m',
  name: 'MobileHome',
  component: () => import('@/views/Home/Mobile.vue')
}

// 桌面端首页
{
  path: '/',
  name: 'Home',
  component: () => import('@/views/Home/index.vue')
}
```

### 自动跳转
路由守卫中实现设备检测和自动跳转逻辑。

## 注意事项

1. **导航一致性**：移动端和桌面端使用相同的导航栏
2. **页脚一致性**：移动端首页也使用完整页脚
3. **无底部导航**：不使用APP样式的底部导航栏
4. **触控优化**：按钮和链接尺寸适合触摸操作
5. **性能优化**：图片懒加载，减少不必要动画
